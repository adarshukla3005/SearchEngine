About this Search Engine

This search engine uses Google search with TF-IDF Validation to find high-quality blogs and articles.

When you search:

  1. The engine enhances your query to focus on blogs and articles
  2. Google search finds relevant results
  3. TF-IDF validation ensures each result is a blog/article
  4. Results are ranked by relevance and displayed with explanations
  5. Pre-crawled blog data is used to supplement Google results

Running the search engine: python app.py

Running the crawler separately: python precrawl.py
